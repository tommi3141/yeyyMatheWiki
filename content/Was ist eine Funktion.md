> [!info] Was ist eine Funktion? 
> Eine Funktion ist eine Zuordnung, die
>- **jedem** Element x aus einer [Menge](https://de.serlo.org/mathe/1981/menge) X (auch [Definitionsbereich](https://de.serlo.org/mathe/1575/definitionsbereich-einer-funktion) genannt)
>- **eindeutig**
>- **ein** Element y einer Menge Y (auch [Wertebereich](https://de.serlo.org/mathe/1655/wertemenge) genannt) **zuordnet**.
>Das Element y wird **Funktionswert an der Stelle** x genannt.
